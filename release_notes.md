# World Time Specialist 1.0.1

## English

### New in 1.0.1

- Improved bilingual coverage across the UI: translated theme names, session names, world-city names and map labels in English and Polish.
- Refined world map behavior: yellow base-zone highlight, better city-label placement, Auckland visibility fixes, responsive scaling and a narrower side panel so the map gets more room.
- Updated map labels to use semi-transparent backgrounds, keeping the exact city marker visible under the label.
- Reworked the live world-city side panel so long headings wrap cleanly and the scrollbar appears only when the content actually overflows.
- Reorganized the universal time converter into a clearer left/right layout: source on the left, target on the right.
- Auto-night mode now switches directly to the high-contrast black theme and supports editable start/stop hours.
- Alarm table improvements: clearer type logic for recurring vs one-shot alarms, better status presentation and stronger visual indicators.
- Added more icons across controls and action buttons for better scanability.
- Session ticking is now globally synchronized: only one tick plays while any session is active, it stops immediately when disabled, and the default tick sample is now softer and less intrusive.
- Installer refresh: the packaged setup is updated for v1.0.1 and release assets include refreshed build scripts and checksums.

### Assets

- `WorldTimeSpecialist-Portable.exe`
- `WorldTimeSpecialist-Setup.exe`
- `build-scripts.zip` (build scripts + NSIS + spec)
- `checksums.txt`

## Polski

### Nowości w 1.0.1

- Rozszerzone pełne tłumaczenia EN/PL w całej aplikacji: nazwy motywów, sesji, miast świata i etykiet na mapie.
- Ulepszona mapa świata: żółte podświetlenie strefy bazowej, lepsze pozycjonowanie etykiet miast, poprawiona widoczność Auckland, responsywne skalowanie i węższy panel boczny, dzięki czemu mapa ma więcej miejsca.
- Etykiety miast na mapie mają teraz półprzezroczyste tło, więc kropka dokładnej lokalizacji pozostaje widoczna pod podpisem.
- Panel „Podgląd miast świata” zawija dłuższe nagłówki i pokazuje pasek przewijania tylko wtedy, gdy treść naprawdę się nie mieści.
- Uniwersalny konwerter czasu został przeorganizowany na czytelny układ: źródło po lewej, cel po prawej.
- Tryb auto nocny przełącza aplikację bezpośrednio na motyw „Czarny Kontrast” i pozwala ustawić własne godziny start/stop.
- Tabela alarmów dostała czytelniejszą logikę rodzaju alarmu (powtarzalny/jednorazowy), lepszy status i mocniejsze wyróżnienia wizualne.
- Dodane ikonki w większej liczbie przycisków i sekcji poprawiają czytelność interfejsu.
- Tykanie sesji jest teraz zsynchronizowane globalnie: niezależnie od liczby aktywnych sesji słychać tylko jedno tykanie, po wyłączeniu zanika natychmiast, a domyślny dźwięk jest subtelniejszy i mniej męczący.
- Odświeżony instalator dla wersji 1.0.1 oraz komplet aktualnych assetów releasu, skryptów build i sum kontrolnych.
